Создание компонента

В прошлом разделе мы создали простой компонент и запустили локальный сервер.

Но пока я не объяснил как все работает.

Начнем с файла index.js (именно он является точкой входа в приложение): 

		import React from 'react';
		import ReactDOM from 'react-dom';            
		import App from './App';    
		ReactDOM.render(<App />, document.getElementById('root'));

Мы импортируем необходимые модули: React,ReactDOM и наш App

Далее вызываем ReactDOM.render и наш компонент встраивается в html элемент с id root.

Далее рассмотрим src/App.js

		import React from 'react';
		import logo from './logo.svg';
		const App = () => {
			return (
				
Всем привет, я компонент App!

			);
		}
		export default App;

В первых 2 строчках мы импортируем сам реакт и логотип приложения (позже можно заменить на свой).

Далее мы создаем сам компонент App в виде фукции.

Такой подход к создания компонента называет реакт-хук.

Вообще, существует 3 способа создания компонента:

  1. React Hook
  2. React class
  3. React createClass

React хук мы уже рассмотрели, осталось класс и createClass

		//класс
		class App extends React.Component{
			render(){
				return (
					<div className="news">
					Компонент App
				</div>
				);
			}
		}

На данный момент лучший способ - это хук.

createClass по сути тот же класс, но для браузеров без поддержки es2015

createClass устарел и был перенес в отдельный модуль.

	//createClass, устаревший код:
	var App = React.createClass(
	{
		render: function()
		{
			return (
				<div className="news">
					Компонент App
				</div>
			);
		}
	});

В нашем курсе используются и классы, и хуки. Устаревшего кода нет)
