Подготовка

Из курса по JS вы знаете, что подключить библиотеку (а ReactJS именно библиотека, а не фрейворк) нужно добавить тег script

Но мы пишем современное приложение на современной библиотеке!

Поэтому мы будем использовать npm пакет create-react-app

Если Вы никогда до этого не слышали про npm, то откройте этот урок из курса про NodeJS

Теперь, когда у Вас установлен NodeJS и npm нужно установить create-react-app

Для этого введите, где my-app это имя Вашего проекта, предлагаю react-test

        npm install create-react-app
        npx create-react-app my-app
    
        your-app-folder
        ├── README.md
        ├── node_modules
        ├── package.json
        ├── .gitignore
        ├── public
        │   ├── favicon.ico
        │   ├── index.html
        │   └── manifest.json
        └── src
            ├── App.css
            ├── App.js
            ├── App.test.js
            ├── index.css
            ├── index.js
            ├── logo.svg
            └── serviceWorker.js

Здорово, теперь у нас есть каркас нашего приложения!

Заполните файл public/index.html

	<!DOCTYPE html>
	<html>
		<head>
			<title>React [RU] Tutorial</title>
		</head>
		
		<body>
			<div id="root"></div>
		</body>
	</html>
