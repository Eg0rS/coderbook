Целые числа

В прошлом уроке мы рассказали о функциях, в том числе о типе аргументов и возвращаемого значения.

Думаю, самое время кратко рассмотреть целые числа в Go более подробно.

В Go есть типы данных для целых чисел и натуральных(безнаковых).

int(8/16/32/64) - целое со знаком, занимает соответсвенно 8,16,32,64 бита.

Диапозон значений: [2**(bits-1),2**(bits-1)], где bits - количество битов

uint(8/16/32/64) - целое без знака, занимает соответсвенно 8,16,32,64 бита.

Диапозон значений: [0,2**bits], где bits - количество битов

Также есть тип byte(синоним uint8), rune(синоним int32)

Также есть типы int и uint

Их размер зависит от платформы и может быть равен 4 или 8 байтам.

package main

import "fmt"
        
func main() {
    var a int8 = 1;
    var b int8 = 2;
    fmt.Println(a+b);
}

Обратите внимание, что в Go нет неявного приведения типов.

Т.е Вы не можете сложить, скажем переменные типа int8 и int32.

Вам придется явно привести тип

package main
    
import "fmt"
            
func main() {
    var a int8 = 1;
    var b int32 = 2;
    fmt.Println(int32(a)+b);
}
