Другие циклы

В ECMASCRIPT 2015 появились новые виды циклов - for/in и for/of

Для начала рассмотрим цикл for/in.

Этот вид цикла перебирает перечисляемые свойства объекта

var prices = {apple:1,pear:1.5,soup:3}
for(price in prices){
console.log(price); //выведет apple,pearl,soup
}

До новой версии стандарта вам бы пришлось использовать обычный for вместе с Object.keys(prices)

Цикл for/of перебирает элементы массива

var sum=0;
var arr = [1,5,0];
for(element of arr){
sum+=element; //прибавляем к сумме элемент массива
}
console.log(sum); //6
