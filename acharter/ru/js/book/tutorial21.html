Классы

Классы так же появились в ES2015.

Вообще что такое класс? Из Вики:

Класс - расширяемый шаблон кода для создания объектов, который устанавливает в них начальные значения (свойства) и реализацию поведения (методы).

В качестве примера можно провести класс человека Human

У человека могут быть различные свойства вроде цвет волос, рост

А методы - это то, что человек млжеь делать. Говорить например

Синтаксис объявления классов:

class Complex{
constructor(real,img){
this.real=real;
this.img=img;
}
plus(num){
return new Complex(this.real+num.img,this.img+num.img);
}
minus(num){
return new Complex(this.real-num.img,this.img-num.img);
}
}

В примере представлен пример объявления класса комплексных чисел.

Класс содержит конструктор constructor, методы plus и минус

Давайте создадим экземпляр класса

const num1=new Complex(1,2);
const num2=new Complex(2,3);
const num3=num1.plus(num2); //(3+5i)
